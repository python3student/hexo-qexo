# print函数的基本使用

``` python
print('xxxx')
print("xxxx")
```
以上代码都可以打印出
``` python
xxxx
xxxx
```
即为引号内的内容被显示在屏幕上

如图
![print](https://tt3.pages.dev/img/print.png)

## 必须指出的是，这里的引号和括号必须是英文状态下输入的，否则会出错
